## 🏛️ 2. 价值链物理学八大公理：仿真舱动力学映射

仿真舱不是静态的可视化工具，而是一台遵循价值链物理定律的**动力学沙盘**。每一项参数与算法开关，都对应着八大公理的动作：

### 公理一（目的论）：最大化有效做功 V 与控制内耗 Q
* **物理公式**：`V = M · Π [ C ⊗ P ⊗ D ]`
* **VCM 映射**：价值链中有效做功 `V` 表现为企业资本回报率 `ROIC` 和齐套交付率 `OTIF` 的最大化；结构性耗散废热 `Q` 表现为呆滞库存与断链停线的总体水平。
* **仿真舱表现**：右侧的实时遙测卡片 `Q (内耗呆滞)`、`OTIF 交付率`、`ROIC 资本回报率` 构成了判决性财务指标。

### 公理二（本质论）：碳基算力破产与硅基代偿必然
* **物理公式**：`Vs* (有效复杂度) << Vs (绝对复杂度)`
* **VCM 映射**：面对多级 BOM、网状替代、共享产能所触发的 `O(N!)` 级排列组合，碳基大脑的沟通与计算带宽注定破产，必须依赖硅基求解器极速推演。
* **仿真舱表现**：点击“启动日内秒级自愈”按钮，瞬间释放一圈强烈的青色电磁脉冲光，代表 `C++` 裸金属求解器在毫秒级内完成数百万种供需匹配推演，将系统从高熵混乱状态强行拉回最优有序解。

### 公理三（方案论）：数字双螺旋 `Π = ⟨D, A⟩`
* **物理公式**：`D` (时空数据容器) ⊗ `A` (动力学算子)
* **VCM 映射**：价值链的数字化绝非“画流程图”，而是用高维数据模型 `D`（容器）与排程算法 `A`（动力学求解器）构成的双螺旋去驾驭物理实相。
* **仿真舱表现**：中央旋转的**蓝色格网线螺旋**代表数据容器 `D`，**紫色螺旋线**代表求解算符 `A`。连结两者的**青色桥梁**即是**供需因果 Pegging (Assignment)**。

### 公理四（能力论）：先验的拓扑解码器与单脑奇点
* **VCM 映射**：系统架构不能由各怀鬼胎的部门委员会分裂构建，必须由具备系统科学与业务痛觉融合的“总架构师单脑”完成底层规则的正交化立法。
* **仿真舱表现**：双螺旋节点嵌入了 Tooltip 交互。悬停在 D/A 节点上，会弹出详细的“本体说明”与“演化推演逻辑”，提示读者系统法则的自洽性。

### 公理五（机制论）：碳硅同构的四阶段 `Π` 循环
* **VCM 映射**：系统进化的循环：自由输入 ➔ 逻辑坍缩 ➔ 协同协奏 ➔ 闭环反馈。
* **仿真舱表现**：改变“需求波动”或“供应延迟”，遥测面板的状态指示灯会实时发生相变：“全局供需实时自愈中”（绿色稳态）➔“算法降级自愈中”（黄色扰动）➔“供需严重失衡 - 触发缺料死锁”（红色耗散）。

### 公理六（路径论）：不可观测即不可控制 (维纳边界)
* **VCM 映射**：控制效力取决于观测边界。若无法实时捕捉微观的库存 ETA、机台状态，宏观计划下达必然悬空。
* **仿真舱表现**：左侧的“物理环境扰动控制器”实时模拟黑天鹅事件（如 Volatility 与 ETA Delay）。滑块的调整实时注入偏差变量 `Δ(t)`，系统必须能实时观测才能执行下一步的算符自愈。

### 公理七（动力论）：双核力场共振与做功夹角 (cos θ)
* **物理公式**：`W = F · S · cosθ`
* **VCM 映射**：如果业务算法 `A` 偏离了底层的物理约束 `D`（即 `cosθ` 趋近于 0），系统就会无效空转，输出巨大的摩擦内耗 `Q`。
* **仿真舱表现**：关闭左侧算法开关（如 SWAP 或 Pegging）时，中央的蓝色 `D` 螺旋与紫色 `A` 螺旋会**强行向左右两侧分裂漂移（偏离值 4.5）**，桥梁碎裂，货运粒子无序乱飞，代表部门脱节、cosθ 降为零、系统有效做功停止，`Q` 飙升至红色警戒区。

### 公理八（进化论）：自指演化与碳硅共生
* **VCM 映射**：当出现系统不完备性死锁时，通过高频自愈算子（硅基自愈）进行消纳；当规则发生相变时，由架构师进行元模型重构。
* **仿真舱表现**：当关闭算法开关导致系统“红屏死锁”（Q 爆表、OTIF 跌破 70%）时，通过点击“日内秒级自愈”或重新打开“SWAP / Allotment”，能瞬间强制将数据和算法双螺旋收拢咬合（偏移值归零），OTIF 瞬间弹回 98% 以上，Q 降到极低。
