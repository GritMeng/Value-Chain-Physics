## 🚀 导入步骤 (Step-by-Step Instructions)

1. **登录 ORCID 账户**
   打开浏览器，访问 [ORCID 个人中心](https://orcid.org/my-orcid) 并登录您的账号。

2. **定位到 Works 区域**
   在您的个人主页中，向下滚动至 **Works**（作品/成果）板块。

3. **选择 Import BibTeX**
   点击 Works 板块右上角的 **Add works**（添加作品），在下拉菜单中选择 **Import BibTeX (.bib)**。

4. **上传 .bib 文件**
   点击 **Choose file**（选择文件），从您的电脑中选择已生成的：
   `H:\系统科学\GritMeng_Research_Outputs.bib`

5. **预览并保存成果**
   * 系统会自动解析出该文件中的 **14 项成果**（包含 2 本书籍/学术专著、3 篇联合研究论文以及 9 项发明专利）。
   * 点击成果列表右侧的 **Save**（保存）按钮，或点击底部的 **Save all**（保存全部）一键导入到您的 ORCID 记录中。
