## 🏛️ 1. 级联齐套上限提取（物理木桶原理）

在多约束 BOM 组替代消纳中，一个成品需求所挂接的多个候选替代组，其单次波次能够配出的最大套数 $\text{KittedQty}$ 严格受限于组内物料在手可用库存的最低短板：
$$\text{KittedQty}\_{g, b} = \min\_{i \in \mathbf{M}\_g} \left( \frac{S\_{g, i}}{U\_{g, i}} \right)$$

---

## 📐 2. 齐套成本张量的非线性拆分

传统软件在齐套时只比对“物料采购单价”，这在面对已存沉淀资产时会造成严重的成本扭曲。系统将每一波次的齐套成本进行**非线性双维度拆分**：

### 1. 消耗存量固有资产价值 (ExistCost)：
*   **数学定义**：代表这一波次齐套，消耗了仓库中**已经买回来的、沉淀在手库存**的物理价值之和。
    $$\text{ExistCost}\_{g, b} = \sum\_{i \in \mathbf{M}\_g} \min\left(\text{KittedQty}\_{g, b} \times U\_{g, i}, S\_{g, i}\right) \times C\_{\text{eff}, i}$$

### 2. 增量新增采购成本 (NewCost)：
*   **数学定义**：代表库存配完后，不足的短板部分**必须立刻触发新采购单、掏出现金**的额外支出价值之和。
    $$\text{NewCost}\_{g, b} = \sum\_{i \in \mathbf{M}\_g} \max\left(0, \text{KittedQty}\_{g, b} \times U\_{g, i} - S\_{g, i}\right) \times C\_{\text{eff}, i}$$

## 📐 3. 全局多准则决策 (MCDM) 偏序关系 $\prec$

系统将全网所有竞争组的所有波次记录 $\mathbf{B}\_{g, b}$ 混合，运行全域格排序偏序关系 $\prec$（$\prec$ 代表“优于”）。
规定 $\mathbf{B}\_x \prec \mathbf{B}\_y$ 当且仅当依次通过以下四级刚性偏序判决门禁：

```
       【 LLC 深度比对 】 ── ( llc_x < llc_y ? ) ──► [ 深度浅者优先 (保交期) ]
                                  │
                                  ▼ (相同)
       【 增量现金采购比对 】 ── ( NewCost_x < NewCost_y ? ) ──► [ 现金支出少者优先 ]
                                  │
                                  ▼ (相同)
       【 消耗在手存量比对 】 ── ( ExistCost_x < ExistCost_y ? ) ──► [ 便宜资产消耗优先 ]
```

1.  **门禁一：拓扑 LLC 深度比对**：
    $$\text{LLC}\_x < \text{LLC}\_y$$
    优先选择 LLC 浅的替代路径。因为层级越浅，意味着在时间轴上的“时移阻尼”偏置越少，提前期风险最低（保交期）。
2.  **门禁二：增量现金采购成本比对**：
    $$\text{NewCost}\_x < \text{NewCost}\_y$$
    在层级相同时，优先选择导致企业**必须掏出额外采购现金最少**的齐套波次（保现金流底线）。
3.  **门禁三：消耗固有资产价值比对**：
    $$\text{ExistCost}\_x < \text{ExistCost}\_y$$
    若新增采购成本也相同，优先选择**消耗库存沉淀资产最便宜**的波次（优先清消呆滞，保护高价值库存）。
