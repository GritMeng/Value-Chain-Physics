# IPC 智能计划与控制系统专利布局与战略规划蓝图 (v3.0)

## 一、 IPC (Intelligent Planning & Control) 物理与系统科学本质

要颠覆传统的 APS（高级计划与排程）与 IBP（整合业务计划）行业，我们必须在系统科学与控制论的第一性原理上，重新定义 **IPC（智能计划与控制）** 的物理本质：
*   **Intelligent (智能)**：是指系统面对供应链无限变异时所表现出的**消纳力与自适应稳态控制力**（Ashby 必要多样性定理）。
*   **Planning (计划)**：是向开放巨系统持续输入信息以抵御无序的**信息负熵泵（Negative Entropy Pump）**。
*   **Control (控制)**：是维持系统运行于最优轨道的**同构自反馈调节器（Conant-Ashby Regulator）**。

## 二、 7 大核心专利集群与 11 篇已完成起草专利映射

为了对整个 IPC 全息抗熵网络实施底层保护，我们已完成起草了 **11 篇** 核心发明专利交底书，物理映射排布于 7 大集群中：

```

  集群一：DOD连续内存与有向超图编译  ===================> 专利 8: LLC 拓扑分级并行展开系统
  
  集群二：时域结合律与时空并行消纳  ===================> 专利 3: 时域分治并行前缀 ATP 系统
  
  集群三：列式内存对账与物理沙箱场景 ===================> 专利 7: OLAP 物理隔离 What-If 沙箱
```

### 1. 集群一：DOD 连续内存与有向超图编译层
*   **定位**：支撑 [I] 智能的底层连续内存通道，消除 OOP 指针跳转造成的 Cache Miss。
*   **已落地专利**：
    *   **专利 8**：`一种基于物料依赖 LLC 拓扑分级的大大规模 MRP 并行消纳调度系统及方法`
        *   *核心创新*：三色 DFS 静态防环与低层码（LLC）并行层级划分，层内 OpenMP 绝对并发零锁扣配。

### 2. 集群二：时域结合律与时空并行消纳层
*   **定位**：时域 ATP 并行前缀扫描数学框架，打破时序因果串行锁依赖。
*   **已落地专利**：
    *   **专利 3**：`一种基于时域并行前缀和分治的同节点ATP供需平衡消纳系统及方法`
        *   *核心创新*：Blelloch 时域前缀求和扫描，以及时域二叉段树区间规约（Surplus/Deficit）结合律半群合并算子。

### 3. 集群三：列式内存对账与物理沙箱场景层
*   **定位**： What-If 极速模拟的数据库物理级零锁隔离技术。
*   **已落地专利**：
    *   **专利 7**：`一种基于嵌入式列式内存数据库物理沙箱的多版本 What-If 协同规划系统及方法`
        *   *核心创新*：`ipc.db` 物理文件级瞬间克隆，以及同进程 C++ DOD 计算层与 DuckDB 的零拷贝 `Appender` 管道。

### 4. 集群四：多级库存优化与虚拟乐观池化层
*   **定位**：打破渠道硬分割、释放池化效应的无锁乐观共享库存。
*   **已落地专利**：
    *   **专利 9**：`一种基于原子级乐观时间戳锁的跨区域/渠道虚拟安全库存池化共享系统及方法`
        *   *核心创新*：两阶段分配协议（历史预留锚定 + 剩余量动态共享）与 `std::atomic` version CAS 无锁库存自旋扣减。
    *   **专利 6**：`一种基于有向图并行段树与交叉替代分配的多级库存优化方法及系统`
        *   *核心创新*：RDC-CDC 供应链网络不确定性级联偏微分传播，与交叉替代组波动余量二次重分摊。

### 5. 集群五：有限产能时空调度与事务回滚层
*   **定位**：微观车间精密排程、高频 CTP 承诺与零动态内存分配回溯。
*   **已落地专利**：
    *   **专利 1**：`一种基于解耦点刚性配额与优先级排程的双层规划系统及方法`
        *   *核心创新*：战术 ITP Allocation ➔ 解耦点 IOP 刚性 Allotment 屏障，64位复合优先级，及同家族内无锁 OTP 前移置换；支持跨企业外包（ODM）及多自建工厂的“中央规则治理、分布式协同执行”资源通道防火墙与多工厂采购比例均衡。
    *   **专利 4**：`一种面向级联排产的零分配内存事务性回滚系统及方法`
        *   *核心创新*：多层级 BOM 级联需求 DFS 尝试扣减，在物理共享内存中使用 `MaterialTrace` 进行微秒级原位逆向加回。
    *   **专利 5**：`一种基于有向无环图及时间戳锁定的时空产能置换方法及系统`
        *   *核心创新*：分时段分桶派程（DBD），设备天级产能乐观锁 CAS 原子扣配，与不损高优交期的 Swap 削峰填谷置换。
    *   **专利 10**：`一种基于线程局部零堆分配回滚栈与时空网格的有限能力排产系统及方法`
        *   *核心创新*：TLS 预分配紧凑 `RollbackAction` 日志物理栈，二维连续时空产能网格，与顺序相关换型矩阵自适应回滚。

### 6. 集群六：多路由维度规划与伴生消纳层
*   **定位**：离散制造及复杂流程式加工中伴生收率与级联替代的运筹最优化。
*   **已落地专利**：
    *   **专利 2**：`一种基于联副产品分级分流消纳的配产控制系统及方法`
        *   *核心创新*：呆滞与延误双惩罚联合约束规划，与多工艺路由（CoproductRoute）凸集正交投影梯度迭代求解。

### 7. 集群七：碳硅共生元认知重整与维纳观测探针层
*   **定位**：满足维纳反馈律的物理孪生探针，与控制论纠偏闭环。
*   **已落地专利**：
    *   **专利 11**：`一种基于数据模型同构双螺旋与自反馈控制的抗熵计划调节系统及方法`
        *   *核心创新*：物理内存中 ODM 与 CDM 的同构螺旋紧密对齐，观测探针时序偏差残差计算，与偏差超标时自动触发沙箱隔离纠偏自反馈抗熵回路。

## 三、 11 篇专利交底书级联协同网络

这 11 篇专利绝非孤立的算法拼凑，而是在数据、逻辑与运行期紧密级联的系统级生态防御网：

1.  **What-If 物理沙箱作为底层物理运行底座**（专利 7）：承载所有上层求解器（专利 1, 2, 3, 5, 8, 10, 11）的并发安全运行，实现零数据库行锁。
2.  **LLC 拓扑层级划分充当并发总框架**（专利 8）：将数百万 SKU 的有向图划分为无依赖的 ranks，分发给 CPU 的 OpenMP 并行计算。
3.  **时域段树前缀扫描充当 SKU 内部消纳内核**（专利 3）：与专利 8 的空间并行结合，构成了高通量三维并发计算大底座。
4.  **双层规划 allotment 与优先级控制大方向**（专利 1）：将战术目标固化为 Allotment 刚性隔离墙，指导微观消纳。
5.  **有限排产时空网格负责设备装配排程**（专利 5）：处理微观机台日历、工作日历等物理约束。
6.  **联副产品求解器处理伴生良率**（专利 2）：当展开到伴生节点时挂起常规 MRP，转为梯度投影计算，结果原位更新。
7.  **无锁乐观 SS 池保护替代料并发分配**（专利 9）：为专利 1, 2, 6 所有的库存分配动作提供 `std::atomic` version CAS 无锁保护。
8.  **多级库存优化算法重算 SS 防御边界**（专利 6）：动态计算方差传播，优化备料决策。
9.  **级联回滚栈与有限能力回滚栈充当原位事务保障**（专利 4 和专利 10）：以零动态内存申请的物理原位加回，保障了高频 DFS 参数搜索的事务洁净性。
10. **自反馈抗熵环路构成控制终极闭环**（专利 11）：探针计算交期残差，超标时利用回滚栈和沙箱动态调整 run_rate 与 SS，重新 Netting 纠偏并强制覆盖基线，阻断系统无序熵增。
